# infant-EEG-MVPA-invariance-project
Analysis toolbox and a reproducible pipeline for preprocessing and time-resolved decoding of EEG data, using EEGLAB and CoSMoMVPA toolboxes and R - MATLAB implementations. Includes pairwise &amp; multi-class decoding (cross-validation scheme),  representational similarity analysis (RSA) &amp; RDM construction, and Bayes-factor analysis.
